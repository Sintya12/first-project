Mengenal Dasar CSS

"CSS adalah kumpulan kode yangan digunakan untuk mendefinisikan desaign dari bahasa markup, salah satunya HTML."

Singkatnya dengan menggunakan CSS kita dapat dengan mudah secara keseluruhan mengubah tampilan halaman web sesuai keinginan kita. File pada CSS berekstensi .css.

Sejarah singkat, Nama CSS didapat dari fakta bahwa setiap deklarasi style yang berbeda dapat diletakkan secara berurutan, yang kemudian membentuk hubungan ayah-anak (parent-child) pada setiap style.

CSS sendiri merupakan sebuah teknologi internet yang direkomendasikan oleh World Wide Web Consortium atau W3C pada tahun 1996. Setelah CSS distandardisasikan, Internet Explorer dan Netscape melepas browser terbaru mereka yang telah sesuai atau paling tidak hampir mendekati dengan standar CSS.
Jika ingin mengetahui lebih spesifik lagi tentang CSS silahkan kunjungi website resminya di https://www.w3.org/Style/CSS/

Penggunaan Kode CSS

CSS datang bersama dengan HTML tag. Untuk menginputkan kode css 3 metode antara lain :

1. Metode Inline Style

Cara menginputkan sebuah kode CSS secara langsung ke dalam tag html dg menggunakan atribut.

inline.html

<!DOCTYPE html>
 <html>
  <head>
   <title>Metode Inline CSS</title>
  </head>
  <body>
   <p style="color: red; font-family: monospace; font-size: 20px;">
     Paragraf ini menggunakan metode inline style css 
   </p> 
  </body> 
 </html>

2. Metode Internal CSS

Disebut juga Embedded Stylesheet digunakan untuk memisahkan kode css dari tag HTML. Kekurangan dari metode ini ialah jika ingin membuat beberapa halaman dan tampilan yang sama, maka setiap halaman akan memiliki kode CSS yang sama juga.
Contoh :

internal.html

<!DOCTYPE html>
 <html>
  <head>
   <title>Metode Internal CSS</title>
   <style type="text/css"> 
    p{
     color: blue;
     font-family: arial;
     font-size: 20px;
    }
   </style>
  </head>
  <body>
   <p>Paragraf ini menggunakan metode internal css</p>
  </body> 
 </html>

3. Metode Eksternal CSS

Digunakan untuk "mengangkat" kode CSS tsb ke dalam sebuah file tersendiri yang terpisah sepenuhnya dari halaman HTML.
Contoh :

eksternal.html

<!DOCTYPE html>
 <html>
  <head>
   <title>Metode Internal</title>
   <!-- atau bisa juga menggunakan tag @import url(â€¦) -->
   <style type="text/css"> 
    h1{
     @import url(css/styles.css);
    }
   </style>
  </head>
  <body>
   <p>Paragraf ini menggunakan metode eksternal css</p>
  </body> 
 </html>

styles.css

h1{
 color: green;
 font-family: sans-serif;
 font-size: 20px;
}
