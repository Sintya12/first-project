Mengenal Dasar Bootstrap 4

"Bootstrap adalah sebuah library framework CSS yg cukup populer bagi kalangan web developer, dibuat khusus untuk bagian pengembangan front-end website."

Bootstrap menyediakan kumpulan komponen class interface dasar yg telah dirancang untuk menciptakan tampilan yang menarik, bersih dan ringan. Dia juga memiliki fitur Grid12 kolom, glyphicon, Bootstrap CSS, komponen Javascript, typography dan komponen lainnya yg bersifat siap pakai.

Salah satu contoh website besar yang menggunakan framework bootstrap ialah twitter, inteface twitter di bangun dengan menggunakan bootstrap. Karena sebenarnya bootstrap dikembangkan oleh developer twitter sendiri, sehingga sering disebut “twitter bootstrap”.

Bootstrap bersifat open source dan menggunakan library dependency seperti HTML5, CSS3, dan jQuery.

Sejarah Singkat Bootstrap

Diciptakan di Twitter oleh Mark otto (@mdo) dan jacob (@fat) pada pertengahan 2010. Sebelum menjadi framework open source, bootstrap dikenal sebagai twitter blueprint.

Awal dirilis pada 19 Agustus 2011 dan telah memiliki lebih dari dua puluh rilisan. Pada versi 2, adanya penambahan fungsionalitas responsif terhadap seluruh framework sebagai stylesheet opsional.

Lalu, disempurnakan lagi pada versi 3. Kemudian bootstrap mengembangkan lagi dan rilis versi Alpha5 yang digunakan saat ini yang menjadi salah satu framework front-end paling populer dan proyek open source di dunia.

Kelebihan dan Kekurangan Bootstrap 4

Kelebihan :

  • Dapat mempercepat waktu proses pembuatan front-end website.
  • Tampilan yang cukup terlihat modern.
  • Tampilan sudah responsive sehingga mendukung segala resolusi, baik itu PC, tablet dan juga smartphone.
  • Website menjadi sangat ringan ketika diakses karena bootstrap dibuat dengan sangat terstruktur.
  • Dibangun dengan LESS (Lenear Style Sheet).
  • Teknologi CSS yang sederhana dan mudah digunakan.
  • Kita bisa mengakses dengan mudah informasi dengan fungsi warna, variabel, dan operasi penggunaan.

Kekurangan :

  • Minim gambar karena bootstrap 4 menggunakan CSS versi 3.
  • Membuat kita menjadi kurang kreatif karena bergantung dengan coding yang dikembangkan oleh developer.

Persiapan Memulai Bootstrap

  • Download file distribusi Bootstrap di https://getbootstrap.com/, extract file tsb. Lalu install, ada 3 perintah untuk install bootstrap :
    1. $ bower install bootstrap
    2. npm install bootstrap
    3. composer require twbs/bootstrap
  • Download jQuery dan popper library, karna bootstrap tidak menyertakan file tersebut didalamnya.
  • Pastikan memiliki code editor.
  • Paham tentang dasar HTML, CSS dan Javascript.
