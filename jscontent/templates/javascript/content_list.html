Mengenal Javascript

"Javascript berguna untuk menyempurnakan sebuah tampilan dan sistem halaman web"

JavaScript dibuat dan didesain oleh Brandan Eich, seorang karyawan Netscape, pada bulan September 1995. Awalnya bahasa pemrograman ini disebut Mocha, kemudian diganti ke Mona, lalu LiveScript sebelum akhirnya resmi menyandang nama JavaScript.

Sejak saat itu, JavaScript terus bertumbuh dan berkembang. Pada akhirnya di tahun 2016, sebanyak 92% website diketahui menggunakan JavaScript.

Javasript adalah sebuah bahasa permrograman yang khusus dirancang untuk website.

Javascript berjalan disisi klien

Artinya penggunaan javascript hanya terbatas pada web-browser saja, tidak dapat melakukan manipulasi data pada sisi server. Sebuah javascript diapit oleh tag <script></script>.

File pada Javascript berekstensi .js, .javascript.

<script language="javascript">
  //javascripy kode disini…
</script> 

Atau 

<script type="text/javascript">
  //javascripy kode disini….
</script>

Penggunaan Kode JS

Sebenarnya sintaks javascript dengan bahasa pemrogramman lainnya itu sama yang membedakan hanya karakteristik sintaks dari setiap bahasa pemrogramman tersebut.
Sebelum menggunakan kode javascript diharapkan mengerti tentang struktur bagian dasar kode javascript, antara lain :

Function()

Function merupakan sebuah blok kode atau beberapa baris kode pada javascript. Function juga merupakan fasilitas disetiap pemrograman untuk membuat sebuah perintah yg dapat digunakan berulangkali.

getElementById('wrap').innerHTML = ;

getElementById('wrap') diambil element yang berdasarkan id-nya &lit;div id="wrap"></div> jadi dapat diubah, tidak harus berId wrap. Lalu pada innerHTML itu dapat berisi tipe data string, tipe data angka, variabel, dll.

Parameter

Parameter adalah sebuah nilai yang akan diletakkan diantara kurung pada function() berguna untuk dapat diolah datanya di dalam fungsi itu sendiri.

Contoh pada file dibawah ini :

input_js.html

<!DOCTYPE html>
<html lang="en" dir="ltr">
 <head>
  <meta charset="utf-8">
  <title>Struktur Kode JS</title>
 </head>
 <body>
  <h4>Struktur Kode JS</h4>
  <!-- Id wrap -->
  <div id="hasil"></div>
  <script type="text/javascript">
   <!--- function --->
   function jsKu(tempat) {
    <!-- Parameter -->
    var x = tempat + "Bali.";
    <!-- Id wrap -->
    document.getElementById("hasil").innerHTML = x ;
   }
   jsKu("Anton dan keluarga pergi ke ");
  </script>
 </body>
</html>

Cara penggunaan sebuah kode javascript ke dalam halaman web hampir sama dengan css, terdapat 2 metode yaitu :

  • Metode Internal, dengan memasukkan secara langsung ke dalam html.

    internal.html
    • <!DOCTYPE html>
    <html lang="en" dir="ltr">
     <head>
      <meta charset="utf-8">
      <title>Penggunaan kode JS</title>
      </head>
     <body>
      <h4>Penggunaan kode JS: Metode internal</h4>
      <div id="wrap"></div>
      <script type="text/javascript">
       function PrintText(nama) {
        var text = nama + "Rio.";
        document.getElementById("wrap").innerHTML = text ;
       }
       PrintText("Laki - laki itu bernama ");
      </script>
     </body>
    </html>

    Output yang ditampilkan

  • Metode Eksternal, dengan membuat file js terpisah.
    Jika menggunakan metode eksternal buat file javascript terpisah. Lalu import file js tersebut pada halaman html.

    eksternal.html
    • <!DOCTYPE html>
    <html lang="en" dir="ltr">
     <head>
      <meta charset="utf-8">
      <title>Penggunaan kode JS</title>
     </head>
     <body>
      <h4>Penggunaan Metode Eksternal JS</h4>
      <div id="wrap"></div>
      <script type="text/javascript" src="../Javascript/scripts.js"></script>
     </body>
    </html>

    scripts.js

    function PrintText(nama) {
      var text = nama + "Talita.";
      document.getElementById("wrap").innerHTML = text ;
    }
    PrintText("Wanita itu bernama ");

    Output yang ditampilkan :
