Mengenal Python

Python dikembangkan oleh Guido van Rossum pada tahun 1990 di Stichting Mathematisch Centrum (CWI), Amsterdam sebagai kelanjutan dari bahasa pemrograman ABC. Versi terakhir yang dikeluarkan CWI adalah 1.2.

Tahun 1995, Guido pindah ke CNRI di Virginia Amerika sambil terus melanjutkan pengembangan Python.

Versi terakhir yang dikeluarkan adalah 1.6. Tahun 2000, oleh Guido dan para pengembang inti Python pindah ke BeOpen.com yang merupakan sebuah perusahaan komersial dan membentuk BeOpen PythonLabs.

Python 2.0 dikeluarkan oleh BeOpen. Setelah mengeluarkan Python 2.0, Guido dan beberapa anggota tim PythonLabs pindah ke DigitalCreations.

Saat ini pengembangan Python terus dilakukan oleh sekumpulan pemrogram yang dikoordinir Guido dan Python Software Foundation.

Python Software Foundation adalah sebuah organisasi non-profit yang dibentuk sebagai pemegang hak cipta intelektual Python sejak versi 2.1 dan dengan demikian mencegah Python dimiliki oleh perusahaan komersial.

Saat ini distribusi Python sudah mencapai versi 2.7.14 dan versi 3.7.4

Apa itu Python ?

"Python is interpreter language."

Artinya python ini hanya bekerja pada mengeksekusi kode program (yang dibuat oleh programmer) lalu menerjemahkannya dalam bentuk bahasa mesin sehingga mesin melakukan instruksi yang diminta oleh programmer tersebut.

Jadi, bisa kita simpulkan bahwa cara kerja bahasa pemrogramman python yaitu :

  1. Skrip dibuat,
  2. Skrip tersebut diterjemahkan ke dalam kode binner oleh (interpreter) python. Agar komputer mengerti perintah yang diberikan.
  3. Komputer menjalankan perintah tersebut.

Cara Installasi Python Untuk Pengguna Ubuntu 18.04

Python mendukung semua sistem operasi, bahkan untuk sistem operasi Linux, hampir distronya menyertakan Python didalamnya.

Jika kita menggunakan sistem operasi linux, ini langkah untuk installasi :

  1. Kunjungi link http://python.org
  2. Buka terminal dengan tekan Ctrl+Alt+T, lalu masukkan perintah seperti ini kemudian tekan Enter :
    $ sudo apt-get install libreadline-gplv2-dev libncursesw5-dev libssl-dev libsqlite3-dev tk-doc libgdbm-dev libc6-dev libbz2-dev
    Jika terdapat error pada package, maka lakukan hal ini :
    • Buka synaptic package manager, bila belum punya segera install karena ini sangat penting.
    • Lakukan update atau install package yang error tersebut.
    Jika kalian tiba – tiba mendapatkan notifikasi seperti ini : 
    E: Could not get lock /var/lib/dpkg/lock-frontend - open 
       (11: Resource temporarily unavailable)
    E: Unable to acquire the dpkg frontend lock (/var/lib/dpkg/
       lock-frontend), is another process using it?
    maka lakukan hal ini :
    • Cari proses apt yang sedang berjalan dengan menginputkan perintah ini kedalam terminal :
      $ ps | grep -i apt
    • Bila sudah ketemu, lakukan pemberhentian proses yang sedang berjalan tersebut dengan menginputkan perintah seperti ini :
      $ sudo kill -9 PID (Proses ID)
    • Lock files sendiri digunakan untuk mencegah adanya dua proses apt yang berjalan bersamaan di sistem yang sama. Kita bisa memperbaikinya dnegan menghapus lock files tersebut dan mengkonfigurasi ulang package dpkg.
  3. Masukan perintah $ ./configure
  4. Masukkan perintah $ make setelah itu $ sudo make install
    Jika tiba – tiba kalian mendapatkan notifikasi seperti ini : 
    Error No Module Named '_ctypes'
    Maka lakukan install package libffi-dev pada synaptic package manager.
  5. Setelah berhasil periksa ulang kembali, dengan memasukkan beberapa perintah seperti ini pada terminal :
    $ sudo make install
    $ which python3.7 (versi yang telah diinstall)
    $ which pip3.7 (versi yang telah diinstall)

  • Masukkan perintah python maka output yang dikeluarkan :
