Mengenal Dasar HTML

HTML atau Hypertext Markup Languange dibuat oleh Tim Bernes-Lee (Seorang ahli Fisika) saat bekerja di CERN (Organisasi Nuklir di Eropa). HTML adalah sebuah yang bahasa markup untuk membuat sebuah halaman web, menampilkan berbagai informasi di dalam sebuah browser.

Kenapa HTML penting dipelajari ?

Karena html merupakan bahan dasar dalam pembuatan sebuah web.

Istilah yang harus dipelajari dalam HTML :

Dokumen HTML adalah kumpulan teks sederhana yang terdiri dari konten halaman dan tag halaman untuk menandai konten tersebut agar dapat dibaca oleh browser.

Elemen HTML adalah konten halaman yang sudah ditandai oleh tag HTML.
Ekstensi File pada HTML yaitu .html.

Tag, bisa kita katakan sebagai sebuah penanda. Penanda awalan dan akhiran sebuah elemen. Tag diapit dengan kurung siku (<...>), seperti tag <html>...</html>.

Atribut adalah bagian dari elemen yang melengkapi elemen tersebut. Atribut tersebut memiliki ciri-ciri atau nilai. Contoh : atribut href yang berfungsi menentukan alamat link dari elemen a .
